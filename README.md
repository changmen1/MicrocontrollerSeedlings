# 基于单片机的温室育苗系统

## 原创不易，如果这个项目可以帮到你请给我一个start，或者有不懂的地方也可以找我，无偿解决
### 插入串口设备，系统会自动读取到设备名称，设置波特率就可以连接了，可以检测温湿度，设置温湿度筏值，也可以自定义调试，直接输入字符串类型就可以，主进程会自动转换的
